# CExe

Repositorio colaborativo para prácticas y ejercicios sencillos en *C*

![GIF divertido de Tenor](https://media.tenor.com/-VqQIA-3LvsAAAAj/gatos-memes-gato-meme.gif)

## Descripción

CExe es un proyecto creado con el propósito de facilitar la colaboración entre estudiantes de primer semestre en programación con C. Aquí encontrarás ejercicios básicos, ejemplos y prácticas que pueden ayudar a reforzar los conceptos fundamentales del lenguaje.

Este repositorio busca:
- Proveer un espacio común para compartir soluciones a ejercicios.
- Promover el trabajo colaborativo y el aprendizaje entre compañeros.
- Servir como referencia y apoyo durante el curso.

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Jmahyals/CExe.git
   ```
2. Accede a la carpeta del proyecto:
   ```bash
   cd CExe
   ```
3. Compila el/los archivos C (requiere GCC o similar):
   ```bash
   gcc -o ejercicio ejemplo.c
   ```
   _Cambia `ejemplo.c` por el archivo que quieras compilar._

## Uso

Ejecuta el programa compilado:
```bash
./ejercicio
```
_Cambia `ejercicio` por el nombre del ejecutable generado._

## Estructura del Proyecto

```
CExe/
├── ejercicios/
│   ├── ejemplo1.c
│   ├── ejemplo2.c
│   └── ...
├── README.md
└── ...
```
_La carpeta `ejercicios/` contiene los diferentes archivos de prácticas colaborativas._

## Contribuciones

¡Todas las contribuciones son bienvenidas! Para colaborar:

1. Bifurca este repositorio.
2. Crea una rama (`git checkout -b nueva-practica`).
3. Sube tu ejercicio nuevo o mejora uno existente.
4. Envía un pull request para revisión.

Recuerda comentar tu código y seguir buenas prácticas de programación.
