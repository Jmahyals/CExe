Crea un programa que calcule el precio final de un producto después de aplicar un descuento. El programa debe:

1. Solicitar al usuario:
   · El precio original del producto (puede tener decimales)
   · El porcentaje de descuento a aplicar (número entero)
2. Calcular:
   · El monto del descuento
   · El precio final después del descuento
3. Mostrar:
   · Un resumen con todos los cálculos
   · Un mensaje especial si el precio final es menor a $100
4. Requisitos adicionales:
   · Validar que el descuento no sea mayor al 50% (usando operadores de comparación)
   · Si el usuario ingresa un descuento mayor al 50%, mostrar un mensaje de advertencia pero aún así calcularlo
   · Usar operadores lógicos para determinar si el precio final es "económico" (menor a $100) Y el descuento fue significativo (mayor a 30%)
